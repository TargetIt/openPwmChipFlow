# PWM 控制器设计规格文档

## 1. 模块概述

| 项目 | 说明 |
|------|------|
| 模块名称 | `pwm_ctrl` |
| 功能描述 | 8 位脉宽调制（PWM）信号生成器 |
| 设计复杂度 | ~25 行 Verilog，无状态机 |
| 核心架构 | 8 位计数器 + 比较器 |

---

## 2. 接口定义

### 2.1 端口列表

| 端口 | 方向 | 位宽 | 类型 | 说明 |
|------|------|------|------|------|
| `clk` | input | 1 | wire | 系统时钟，所有逻辑在上升沿触发 |
| `rst` | input | 1 | wire | 同步复位，高有效 |
| `duty` | input | 8 | wire | 占空比设置值，范围 0–255 |
| `pwm_out` | output | 1 | wire | PWM 输出，组合逻辑驱动 |

### 2.2 端口时序图

```
clk     : ‾‾|__|‾‾|__|‾‾|__|‾‾|__|‾‾|__|‾‾|__|‾‾|__|‾‾|__
rst     : ‾‾‾‾‾‾‾‾‾‾|_____________________________________________
duty    :        XXXX|  0x80 (128)
counter :          00| 01 | 02 | 03 | ... | 7F | 80 | 81 | ... | FF | 00 |
pwm_out :          __|‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|___________________|‾‾‾
                      <--- duty=128: 高电平 --->  <--- 低电平 --->
```

## 3. 功能描述

### 3.1 计数器逻辑

- 8 位无符号计数器 `counter`，范围 0–255
- 每个 `clk` 上升沿递增 1
- 溢出后自动回绕到 0（利用 8 位自然溢出，无需额外逻辑）
- 同步复位时清零：`rst` 为高时，`counter <= 0`

### 3.2 比较器逻辑

- 组合逻辑输出：`pwm_out = (counter < duty)`
- `counter < duty` → `pwm_out = 1`（高电平）
- `counter >= duty` → `pwm_out = 0`（低电平）

### 3.3 占空比行为

| duty 值 | 占空比 | 行为 |
|---------|--------|------|
| 0 | 0% | `pwm_out` 恒为低电平 |
| 1 | 0.39% | 每 256 周期输出 1 周期高电平 |
| 64 | 25% | 每 256 周期输出 64 周期高电平 |
| 128 | 50% | 每 256 周期输出 128 周期高电平 |
| 192 | 75% | 每 256 周期输出 192 周期高电平 |
| 255 | 99.6% | 每 256 周期输出 255 周期高电平 |

> 注意：`duty=255` 时无法达到 100% 占空比，因为 `counter=255` 时 `counter < 255` 为假。如需 100% 占空比，需在上层逻辑处理。

## 4. 时序规格

### 4.1 时钟参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 目标时钟频率 | 50 MHz | 时钟周期 20 ns |
| PWM 输出频率 | ~195 kHz | 50 MHz / 256 |
| PWM 输出周期 | ~5.12 μs | 256 × 20 ns |

### 4.2 延迟特性

| 路径 | 类型 | 说明 |
|------|------|------|
| `clk` → `counter` | 寄存器路径 | 1 个时钟周期延迟 |
| `counter`/`duty` → `pwm_out` | 组合路径 | 8 位比较器延迟（< 2 ns） |

### 4.3 复位时序

- 复位类型：同步复位
- 复位极性：高有效
- 复位响应：下一个 `clk` 上升沿生效
- 复位后状态：`counter = 0`，`pwm_out = 0`（当 `duty > 0` 时 `pwm_out` 在复位释放后下一周期变为 1）

## 7. RTL 参考实现

```verilog
module pwm_ctrl (
  input        clk,
  input        rst,
  input  [7:0] duty,
  output       pwm_out
);
  reg [7:0] counter;

  always @(posedge clk) begin
    if (rst)
      counter <= 8'd0;
    else
      counter <= counter + 8'd1;  // 自动溢出回绕
  end

  assign pwm_out = (counter < duty);

endmodule
```

## 8. 设计限制与已知约束

| 项目 | 说明 |
|------|------|
| 无 100% 占空比 | `duty=255` 时占空比为 99.6%，非 100% |
| 无使能控制 | 没有 `enable` 端口，上电即开始输出 |
| 无分频功能 | PWM 频率固定为 `clk / 256` |
| 无死区时间 | 无互补输出与死区控制 |
| 占空比实时生效 | `duty` 变化立即影响输出，无同步机制 |

> 以上限制均为有意的设计选择，目的是保持设计极简，将精力集中在学习全流程工具链上。
