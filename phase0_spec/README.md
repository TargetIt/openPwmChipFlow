# Phase 0 — 规格与需求定义

## 概述

本阶段在 RTL 编写之前，完成 PWM 控制器芯片的需求分析与设计规格定义。所有后续阶段（RTL、仿真、综合、布局布线、物理验证、GDS 输出）均以本阶段文档为基准。

## 文档列表

| 文档 | 文件名 | 说明 |
|------|--------|------|
| 需求文档 | [requirements.md](requirements.md) | 项目背景、功能需求、性能指标、约束条件 |
| 设计规格文档 | [design_spec.md](design_spec.md) | 模块接口、功能描述、时序规格、实现约束 |

## 目录结构

```
phase0_spec/
  ├── README.md            ← 本文档
  ├── requirements.md      ← 需求文档
  └── design_spec.md       ← 设计规格文档
```

## 检查清单

- [ ] 需求文档完成 — 功能需求、性能需求、约束条件明确
- [ ] 设计规格文档完成 — 接口定义、功能描述、时序规格清晰
- [ ] 需求与设计规格一致 — 无遗漏、无冲突
- [ ] 评审通过 — 团队确认文档可作为后续开发基准
