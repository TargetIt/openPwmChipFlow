# Phase 1 — RTL 编写

## 概述

本阶段完成 PWM 控制器的 RTL 设计。PWM（脉宽调制）控制器由一个 8 位计数器和一个比较器组成，逻辑简单清晰。

## 模块规格

### 端口定义

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `clk` | input | 1 | 时钟 |
| `rst` | input | 1 | 同步复位，高有效 |
| `duty` | input | 8 | 占空比 0–255，对应 0%–100% |
| `pwm_out` | output | 1 | PWM 输出信号 |

### 工作原理

1. 8 位计数器 `counter` 在每个时钟上升沿自增 1，溢出后自动回绕到 0
2. 当 `counter < duty` 时，`pwm_out` 输出高电平
3. 当 `counter >= duty` 时，`pwm_out` 输出低电平
4. 同步复位 `rst` 高有效时，计数器清零

### 设计要点

- 代码控制在 25 行以内，不过度设计
- 使用同步复位，便于综合
- 8 位计数器自动溢出回绕，无需额外逻辑
